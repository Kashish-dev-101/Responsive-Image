Art Direction using picture element

This demo uses the picture element to serve completely different images based on viewport width.

Resize the browser or use Chrome device emulator to see the image change between mobile and desktop.

Read more on how the picture element works in our Responsive Image Guide.

Mobile: Portrait image optimized for narrow screens

Desktop: Landscape image optimized for wide screens

Key difference:
srcset + sizes selects different resolutions of the same image.
pictureenables art direction by selecting completely different images.
